    - REST API
    - GIT

## Configuration
See the file `.env.example` to configure the environment variables.

## Installation
```bash
npm install
```

## Running
```bash
npm start
```

## Documentation
This project uses Swagger to document the API. 
To access the documentation, run the project and access the URL `http://localhost:5000/docs`.
